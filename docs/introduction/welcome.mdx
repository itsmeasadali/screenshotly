# Welcome to Screenshotly

Screenshotly is a powerful API service that allows you to capture high-quality screenshots of any webpage, customize them with mockups, and integrate them into your applications.

## Features

- **High-Quality Screenshots**: Capture full-page or specific element screenshots with customizable viewport sizes
- **Device Mockups**: Apply beautiful device frames to your screenshots
- **Flexible Output**: Get your screenshots in PNG, JPEG, or PDF formats
- **Pro Features**: AI-powered element removal and smart cleaning capabilities

## Getting Started

To start using Screenshotly:

1. [Sign up](/sign-up) for a free account
2. Get your API token from the [dashboard](/dashboard/tokens)
3. Follow our [quickstart guide](/introduction/quickstart)

## Example

Here's a quick example of how to capture a screenshot:

```bash
curl -X POST https://api.screenshotly.app/screenshot \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com",
    "device": "desktop",
    "format": "png"
  }'
```
