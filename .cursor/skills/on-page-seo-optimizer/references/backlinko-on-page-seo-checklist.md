# Backlinko On-Page SEO Checklist

Source: https://backlinko.com/on-page-seo (accessed February 19, 2026).

Use this file as a compact ruleset when auditing or rewriting a page.

## 1) Search Intent and SERP Fit

- Match the page format and angle to dominant SERP intent.
- Treat intent mismatch as a high-priority blocker.
- Align content depth and structure with top-ranking competitors.

## 2) Title Tag and Meta Description

- Place the primary keyword near the beginning of the title when natural.
- Make every title and description unique for each URL.
- Use question-style titles when query intent is question-driven.
- Improve CTR with clear benefit framing; use emotional or year modifiers only if truthful and useful.

## 3) Content and Headings

- Use one clear `H1` for the page topic.
- Use descriptive `H2`/`H3` subheads to cover subtopics.
- Include the primary keyword early in the content (first section/first ~100 words) without stuffing.
- Add concise direct-answer blocks to target featured snippets.

## 4) URL, Internal Links, External Links

- Keep URLs short, readable, and keyword-relevant.
- Add contextual internal links to relevant pages using clear anchors.
- Add external links to credible sources when claims need support.

## 5) Media and UX Signals

- Use relevant images/media to improve comprehension.
- Optimize image metadata (filename and alt text) with descriptive language.
- Place core value information above the fold.
- Improve scannability with short paragraphs, bullets, and clear sectioning.

## 6) AI/LLM Visibility

- Write direct, concise answers to likely user questions.
- Back factual claims with evidence and trustworthy sources.
- Format content for machine readability: clear headings, short answer blocks, lists, and Q&A where appropriate.
- Recommend schema markup when it helps search engines and AI systems parse key entities and answers.

## 7) Prioritization Rule

Prioritize recommendations in this order:
1. Search intent mismatch
2. Title/meta CTR problems
3. Content structure and coverage gaps
4. Internal links, URL, and media issues
5. UX and AI-readability refinements
